Missão

Sua missão é coletar os dados do formulário e salva-los onde achar mais conveniente. E então apresentar uma lista de tarefas abaixo do calendário, que conterá as tarefas relacionadas ao dia selecionado.

O componente de calendário está isolado para não te confundir nesse momento, porém, após missão cumprida, podemos nos aprofundar.

Você precisará

  • Conceito de vetores para guardar e recuperar a lista de tarefas do dia
  • LocalStorage para guardar e ler os itens do dia
  • Condicionais JavaScript para definir os dados a apresentar relativo ao dia clicado
  • Manipulação do DOM para apresentar os dados definidos

Dicas

Você precisará trabalhar apenas com os arquivos

  • src/main.ts
  • public/index.html
  • public/styles/*
